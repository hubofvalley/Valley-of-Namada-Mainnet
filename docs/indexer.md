# Indexer Guide

Deploy Namada Indexer and MASP Indexer services.

## Namada Indexer

The Namada Indexer provides blockchain data indexing for queries.

### Installation

1. Launch Valley of Namada:
   ```bash
   bash <(curl -s https://raw.githubusercontent.com/hubofvalley/Mainnet-Guides/main/Namada/resources/valleyofNamada.sh)
   ```
2. Select **"Indexer"** → **"Deploy Namada Indexer"**
3. Follow the prompts

### Current Version

| Component | Version |
|-----------|---------|
| Namada Indexer | v4.1.0 |

## MASP Indexer

The MASP Indexer provides indexing for Multi-Asset Shielded Pool transactions.

### Installation

1. Launch Valley of Namada
2. Select **"Indexer"** → **"Deploy MASP Indexer"**
3. Follow the prompts

### Current Version

| Component | Version |
|-----------|---------|
| MASP Indexer | v1.4.7 |

## Indexer Snapshots

### Namada Indexer Snapshot

1. Launch Valley of Namada
2. Select **"Indexer"** → **"Apply Namada Indexer Snapshot"**

### MASP Indexer Snapshot

1. Launch Valley of Namada
2. Select **"Indexer"** → **"Apply MASP Indexer Snapshot"**
