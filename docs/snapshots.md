# Snapshots Guide

Apply snapshots for faster node synchronization on Namada mainnet.

## Validator Node Snapshot

1. Launch Valley of Namada:
   ```bash
   bash <(curl -s https://raw.githubusercontent.com/hubofvalley/Mainnet-Guides/main/Namada/resources/valleyofNamada.sh)
   ```
2. Select **"Node Interactions"** → **"Apply Snapshot"**
3. Follow the prompts

## Indexer Snapshots

### Namada Indexer Snapshot

1. Launch Valley of Namada
2. Select **"Indexer"** → **"Apply Namada Indexer Snapshot"**

### MASP Indexer Snapshot

1. Launch Valley of Namada
2. Select **"Indexer"** → **"Apply MASP Indexer Snapshot"**
