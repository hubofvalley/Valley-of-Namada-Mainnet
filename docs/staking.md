# Staking Guide

Stake and delegate NAM tokens on Namada mainnet.

## Delegation

1. Launch Valley of Namada:
   ```bash
   bash <(curl -s https://raw.githubusercontent.com/hubofvalley/Mainnet-Guides/main/Namada/resources/valleyofNamada.sh)
   ```
2. Select **"Staking Interactions"** → **"Stake Tokens"**
3. Choose:
   - **Delegate to Grand Valley**
   - **Self-delegate**
   - **Delegate to another validator**
4. Enter wallet name and amount

## Unstaking

1. Launch Valley of Namada
2. Select **"Staking Interactions"** → **"Unstake Tokens"**
3. Enter validator address and amount

## Claiming Rewards

1. Launch Valley of Namada
2. Select **"Staking Interactions"** → **"Claim Rewards"**

## Cubic Slashing Rate Check

Check cubic slashing rates:

1. Launch Valley of Namada
2. Select **"Utils"** → **"Cubic Slashing Calculator"**

## RPC Options

When performing staking operations, you can choose:
- **own** - Use your own node's RPC
- **gv** - Use Grand Valley's RPC (default)
